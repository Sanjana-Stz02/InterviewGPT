# InterviewGPT

AI-Powered Interview Preparation Platform built with MERN Stack and NLP.

## Overview

InterviewGPT helps job seekers prepare for interviews by analyzing resumes, comparing them against job descriptions, identifying skill gaps, generating interview questions, conducting mock interviews, and providing AI-powered feedback.

## Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Protected Routes

### Resume Analysis
